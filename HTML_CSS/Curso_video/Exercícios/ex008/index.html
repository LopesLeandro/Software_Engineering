Negrito / Destaque

Jeito não semântico:

Nessa frase temos um termo em negrito usando a tag <b>

Nessa frase temos um texto em itálico não semântico, usando a tag <i>

Jeito semântico de ser:

Nessa frase temos um termo em negrito usando a tag <strong>

Nessa frase temos um texto em destaque no jeito semântico de ser, usando a tag <em>

Outros jeitos de marcar texto com base no pdf.

Você sabe usar marca texto? Basta usar a tag <mark>.

Você pode mudar as cores das marcações, mas é necessário usar CSS dentro da tag <style> direto na linha ou dentro das tags <HEAD>

Texto deletado, basta usar a tag <del>

Texto inserido, que é o exato oposto do anterior, usamos a tag <ins>.

E texto sobrescrito e subscrito, como os casos de x2 ou H2O? Para isso usamos as tags <sup> ou <sub> respectivamente.

Citações

Criar uma citação em HTML, podemos usar a tag <q> (do Inglês quote, que significa citar). O texto que estiver entre <q> já vai receber automaticamente as aspas, mas não terá nenhum deslocamento. Essa técnica é mais usada quando queremos uma citação no meio de um parágrafo.

Também podemos criar citações mais longas (em bloco) e que tenham um parágrafo só para si. Nesse caso, colocaremos tudo dentro de <blockquote> e o texto ganha um recuo automaticamente. Podemos também colocar um link para o texto original, usando o parâmetro cite dentro da tag.
